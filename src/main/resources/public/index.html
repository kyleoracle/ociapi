fn workshop automation

  • By a single click, batch create workshop instances in minutes
  • Workshop required libs are pre-installed. e.g. docker/fn server
  • End-to-end workshop script is pre-installed, just issue ./demo.sh

Steps

  1. create instance
  2. list provisioning instance
  3. list running instance
  4. connect to instance
  5. run the demo 
    		sudo -i
		./start.sh
		./demo.sh
		./stop.sh
  6. delete instance
